# Python
A collection of beginner Python programs including password validation, report card generation with percentage calculation, loan eligibility checking, and driving license age verification using basic programming concepts.

# Basic Python Programs Collection

It is designed to practice fundamental programming concepts such as user input, calculations, conditional statements, and logical decision-making.

This repository includes simple real-world based projects:

**Password Checker** – Validates passwords based on basic security rules.

**Report Card Generator** – Takes student marks as input, calculates the total marks and percentage, determines the remarks, and displays a complete report card.

**Loan Eligibility Checker** – Checks whether a user qualifies for a loan based on given conditions.

**Driving Licence Age Checker** – Determines eligibility for a driving licence based on age requirements.
